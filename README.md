# Informare

Welcome to Informare, a web application designed for educational and informational purposes.

## Overview

The Informare web application offers a variety of features, including a home page, patch notes, homework management, voting, daily messages, and more. Users can customize the theme and account settings to enhance their experience.

## Files

- `index.html`: Main HTML file containing the structure of the web application.
- `inform.css`: Stylesheet for styling the web application.
- `photos/`: Folder containing images used in the application.
- `scripts/`: Folder containing JavaScript files for various functionalities.

## Features

1. **Announcement Pop-Up**: A pop-up displaying the latest updates and highlights.
2. **Navigation Bar**: Navigate through different sections of the application, including Home, Patch Notes, Homework, Voting, and Daily Message.
3. **Theme Customization**: Choose between light, custom, and night modes. Additionally, a custom color mode is available.
4. **Profile Management**: Access account settings, including the option to upload a profile picture.
5. **Main Page**: Display key information such as menu items and grades.
6. **Lunch Schedule**: View the lunch schedule in a user-friendly calendar format.
7. **Login Functionality**: Users can log in for enhanced access to features.

## Usage

1. Open `index.html` in a web browser to access the Informare web application.
2. Explore different sections using the navigation bar.
3. Customize themes and account settings as desired.
4. Check the lunch schedule for daily information.

## Additional Notes

- The web application is designed for educational purposes and may include features such as homework management and daily messages.
- Users can customize the application's theme to enhance their visual experience.
- Account settings allow users to manage their profile picture and other preferences.

Feel free to explore and customize the web application to meet your specific needs.
